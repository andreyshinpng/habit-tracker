# Трекер привычек

Минималистичное веб-приложение для отслеживания ежедневных привычек.

## Запуск

```bash
docker-compose up -d
```

Приложение будет доступно по адресу: http://localhost:8080

## Остановка

```bash
docker-compose down
```

## Обновление существующей базы данных

Если приложение уже запущено и нужно добавить поддержку перетаскивания привычек:

```bash
docker exec -i habit-tracker-mysql mysql -uroot -proot123 habit_tracker < migrate.sql
```

## Функционал

- Добавление и удаление привычек (soft delete)
- Отметка выполнения по дням
- Выбор месяца и года для просмотра
- Интерактивная таблица с визуализацией прогресса
- Изменение порядка привычек перетаскиванием (drag and drop)

## Стек технологий

- PHP 8.2 + Apache
- MySQL 8.0
- Vanilla JavaScript
- Docker
